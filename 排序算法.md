
# 元素排序算法

----------


## 1. 插入排序

算法轮廓：

    insertionSort(data[]){
	    for(i=1;i<data.length;i++)
	    tmp = data[i];
	    将所有大于tmp的元素data[j]都移动一个位置；
	    将tmp放到正确的位置上；
	 }
优点： 只在需要的时候对数组排序。

时间复杂度：介于$O(n)$和$O(n^2)$之间，接近后者。


----------


## 2. 选择排序

算法轮廓：

    selectSort(data[]){
	    for(i=0;i<data.length;i++)
	    从data[i], ..., data[data.length-1]中选取最小的元素；
	    将它和data[i]交换；
	}

----------

## 3. 冒泡排序

算法轮廓：

    bubbleSort(data[]){
	    for(i=0;i<data.length;i++)
		    for(j=data.length-1;j>i;--j)
		    如果顺序错误就交换j和j-1的位置上的元素；
	}
